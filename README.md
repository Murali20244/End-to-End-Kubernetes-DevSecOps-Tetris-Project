This project demonstrates a complete **DevSecOps pipeline** for deploying a containerized Tetris application on **AWS EKS**, integrating CI/CD, GitOps, security scanning, and Infrastructure as Code.

---

## 📂 Project Structure

1. **EKS-TF** — Terraform scripts for provisioning AWS EKS clusters  
2. **Jenkins-Pipeline-Code** — Jenkins CI/CD pipeline for automated builds and deployments  
3. **Jenkins-Server-TF** — Terraform configuration for provisioning Jenkins on EC2  
4. **Manifest-file** — Kubernetes manifests for deploying the Tetris application  
5. **Tetris-V1** — Initial version of the Tetris application  
6. **Tetris-V2** — Enhanced version with improvements
